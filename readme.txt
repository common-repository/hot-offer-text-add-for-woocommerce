By using this plugin, You will be able to add Ads/Information/Image to various positions in Woocommerce single product page. 

== Description ==

Show Ads in  Woocommerce is very helpful to add Ads in many formats like Text/HTML/Image to product(s). Ads will show in  selected position. 
Following position are available in  'Show Ads in  woocommerce'.

14 Positions are-

*Before Title

*After Title

*Before Excerpt

*After Excerpt

*Before Meta

*After Meta

*Before Sharing

*After Sharing

*Before Product Summery Tab

*After Product Summery Tab

*Before Upsell Display

*After Upsell Display

*Before Related Products

*After Related Products




== Installation ==

How to insall, active and configure-

1. Upload the plugin files to the `/wp-content/plugins/Hot-Offer-Text-Add-for-Woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Woocommerce Ads' in Left Menu to configure the plugin
